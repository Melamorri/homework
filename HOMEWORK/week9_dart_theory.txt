1. Зачем программистам нужны библиотеки?
    Библиотеки – это уже готовые решения определённых задач, это позволяет экономить время при
    написании кода.

2. Найдите 3 различные открытые библиотеки Dart и их назначение
    libphonenumber - проверяет на валидность телефонные номера, по разным странам и регионам;
    custom_check_box - для создания чекбоксов;
    fluttertoast - пакет для всплывающих сообщений.

3. Как можно добавлять зависимости в приложение?
    Можно добавить зависимость в файл pubspec.yaml. Можно добавить библиотеку через командную строку
    и файл сам обновится, нужно будет только подтянуть зависимости.

4. Как можно узнать устаревшие зависимости?
    С помощью команды dart pub outdated.

5. Где хранятся данные об установленных версиях пакетов?
   В файле pubspec.lock

6. Какую информацию может содержать файл pubspec?
    Зависимости, которые необходимы для проекта, например шрифты, картинки, различные библиотеки.
