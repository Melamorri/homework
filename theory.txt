1. Что такое база данных?
    Набор упорядоченных данных хранящийся в структурированном виде.

2. Чем отличается SQL база данных от NoSQL?
    SQL-база более стандартизирована: требует определённой структуры хранения данных, использования
    языка SQL, тогда как NoSQL-база не требует таких ограничений.

3. Какие типы данных имеет SQLite?
    NULL: отсутствие значения
    INTEGER: целое число
    REAL: число с плавающей точкой
    TEXT: строка текста
    BLOB: бинарные данные

4. Какую команду нужно вызвать, чтобы запустить кодогенерацию?
    flutter packages pub run build_runner build или
    flutter packages pub run build_runner build --delete-conflicting-outputs
